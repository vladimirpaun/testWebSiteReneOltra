Exporter en HTML

Le contexte WebCamp permet d’obtenir l'ensemble des informations des établissements autorisés pour votre compte.
Cette méthode retourne toutes les informations d'un établissement : Coordonnées, critères, mode de paiements, photos, etc… Ainsi que l’ensemble des catégories d’hébergements et tarifs associés.
En récupérant ces informations vous pouvez construire votre catalogue de produits.

Mapping

NameTypeMandatoryDefaultRepetitionsComment
idSiteInt null Si défini, limite la réponse uniquement à cet établissement.
idUserStringyes

passwordStringyes   
returnSiteOnlyBoolean false Permet de limiter la taille de la réponse.
Si false, retourne également : channel, paiement, customerListInformation.
returnProductBoolean false Retourne ou non les catégories dans la réponse.
Si true, retourne : les catégories et les tarifs des catégories.
returnMediaObjBoolean false Si true, retourne les images et médias. Pour le camping et les catégories.
mediaDataTypemediaDataType data Défini le type de retour des images si returnMediaObj = true

Conseil d’utilisation

Les données changent rarement et la réponse peut être très lourde en fonction des options demandées.
Il ne faut donc pas appeler cette méthode systématiquement mais, par exemple, une fois par jour (la nuit le plus souvent), pour mettre à jour votre catalogue.
La construction de votre catalogue doit se faire en plusieurs appels :

  1. Un appel pour récupérer l’ensemble des établissements (sans détails, sans catégories, sans photos, etc…) afin de connaitre l’ensemble des campings disponibles en utilisant
    • <returnSiteOnly>true</returnSiteOnly>
    • <returnProduct>false</returnProduct>
    • <returnMediaObj>false</returnMediaObj>
  2. Un appel pour chaque établissement disponible en demandant le détail des catégories, photos, etc… En utilisant
    • <idSite>?</idSite>
    • <returnSiteOnly>false</returnSiteOnly>
    • <returnProduct>true</returnProduct>
    • <returnMediaObj>false</returnMediaObj>
